# Sentence Practice

Language learning application for practicing German sentences.
<img width="1664" height="918" alt="image" src="https://github.com/user-attachments/assets/ae9c56ad-8982-4d6a-aaf2-945659ea6b53" />


## Features

- German ↔ English learning modes
- Audio playback
- Sentence ratings
- Review statistics
- Search
- Random mode
- Auto mode
- Parts and levels (A1, A2, etc.)

## Tech Stack

- React
- Node.js
- Express
- PostgreSQL
- Axios
